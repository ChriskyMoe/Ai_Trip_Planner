# Flight Booking API Options

## Recommended: Amadeus API (Best Choice)

**Why Amadeus:**
- ✅ **Free tier available** - $200 free credit/month for development
- ✅ **Comprehensive** - Flight search, pricing, booking
- ✅ **Widely used** - Industry standard
- ✅ **Good documentation** - Well-documented APIs
- ✅ **Real-time data** - Live flight availability and prices

**Get Started:**
1. Sign up: https://developers.amadeus.com/
2. Create app: Get API Key and API Secret
3. Free tier: $200 credit/month (enough for development)

**APIs You'll Need:**
- **Flight Inspiration Search** - Find flight deals
- **Flight Offers Search** - Search flights by route/dates
- **Flight Price Analysis** - Get detailed pricing
- **Flight Create Orders** - Book flights (paid feature)

## Alternative Options

### 1. Skyscanner API (RapidAPI)
- **Pros:** Good coverage, real-time prices
- **Cons:** Requires RapidAPI subscription, limited free tier
- **Get it:** https://rapidapi.com/skyscanner/api/skyscanner-flight-search

### 2. FlightAPI.io
- **Pros:** Simple API, good for basic searches
- **Cons:** Limited booking capabilities
- **Get it:** https://flightapi.io/

### 3. FlightAPIs.com
- **Pros:** Good for travel agencies, instant booking
- **Cons:** More expensive, requires business setup
- **Get it:** https://flightapis.com/
